## Ce qui distingue ce dépôt

Atlas n'est pas qu'un assemblage de projets : c'est une **chaîne de qualité
partagée**, outillée et vérifiable. Voici ce qu'un contributeur y trouve — chaque
point renvoie à la page qui l'atteste.

<CardGrid>
  <Card title="Le bilan, vérifiable" icon="approve-check">
    Toutes les pratiques d'ingénierie en place, **discipline par discipline**,
    chacune rattachée à un mécanisme concret du dépôt — écarts compris. La
    page-pivot pour savoir ce qui est réellement appliqué.

    [Normes et pratiques appliquées →](/atlas/quality/normes/)

  </Card>
  <Card title="Des conventions explicites" icon="open-book">

    [Documentation →](/atlas/quality/documentation/)

  </Card>
  <Card title="Qualité et sécurité outillées" icon="seti:typescript">
