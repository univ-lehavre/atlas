# Scripts

Scripts transverses du monorepo Atlas.

Ce répertoire contient les automatisations qui ne sont pas des packages publiables: audits de structure et de licences, génération d'artefacts de documentation, analyse du dépôt et orchestration de publication. Les scripts sont regroupés par domaine fonctionnel.

- `audit/`: contrôles de structure et qualité de workspace
- `docs/`: génération des artefacts de documentation
- `release/`: scripts liés à la publication
