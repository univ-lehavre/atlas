# Packages

Bibliothèques réutilisables du monorepo Atlas.

Ce répertoire regroupe les packages TypeScript partagés par les applications, CLI et services: clients API, types, validations, erreurs, statistiques, intégrations REDCap, OpenAlex, Appwrite et outils réseau. Chaque sous-répertoire est une unité publiable ou consommable en workspace.
