# @univ-lehavre/atlas-crf-client

Client TypeScript typé pour l'API REDCap.

Ce package fournit un client Effect pour interagir avec REDCap: version, informations projet, instruments, champs, exports et imports de records. Il inclut des types brandés, des erreurs structurées, des adaptateurs par version REDCap et des types générés depuis les spécifications OpenAPI.
