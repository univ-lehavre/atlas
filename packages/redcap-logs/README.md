# @univ-lehavre/atlas-redcap-logs

Collecte et analyse des logs d'audit REDCap.

Ce package récupère les logs de projets REDCap, enrichit les entrées avec les tokens projet, gère le cache, calcule des fenêtres glissantes et des agrégations calendaires mensuelles. Il expose aussi des diagnostics réseau liés aux endpoints analysés.
