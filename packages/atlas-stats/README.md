# @univ-lehavre/atlas-stats

Bibliothèque de collecte et calcul des statistiques Atlas.

Ce package récupère les releases GitHub, les métadonnées npm et les téléchargements de paquets `@univ-lehavre/*`, gère un cache local et calcule les indicateurs consommés par le dashboard et le CLI. Il expose aussi des helpers pour résoudre le workspace, le token GitHub et générer un rapport CLI.
