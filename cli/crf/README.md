# @univ-lehavre/atlas-crf-cli

CLI de diagnostic REDCap et de gestion du serveur CRF.

Le package expose les commandes `crf-api` et `crf-server`. Elles permettent de tester la connectivité REDCap, vérifier les endpoints principaux, produire une sortie humaine ou JSON, et lancer les fonctionnalités serveur liées au service CRF.
