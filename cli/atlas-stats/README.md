# @univ-lehavre/atlas-stats-cli

CLI interactif pour consulter les statistiques GitHub et npm du monorepo Atlas.

La commande collecte ou relit le cache des releases et téléchargements de paquets, calcule des indicateurs par période, puis affiche un rapport lisible en terminal ou exportable en JSON. Elle s'appuie sur `@univ-lehavre/atlas-stats`.
