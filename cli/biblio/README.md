# @univ-lehavre/atlas-biblio-cli

CLI interactif de validation bibliographique OpenAlex.

La commande guide l'utilisateur dans les actions de validation d'un auteur: chargement des stores, sélection d'actions actives, suivi du tableau de bord, gestion des identifiants OpenAlex, affiliations, publications et événements associés. Elle consomme les services de `@univ-lehavre/atlas-validate-openalex`.
