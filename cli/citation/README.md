# @univ-lehavre/atlas-citation-cli

CLI de curation de données chercheur depuis OpenAlex.

La commande accompagne la recherche d'un chercheur, la sélection des formes de nom, des affiliations et des auteurs OpenAlex pertinents, puis la récupération des publications associées. Elle s'appuie sur les fonctions de recherche et de regroupement de `@univ-lehavre/atlas-citation`.
